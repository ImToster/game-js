# Игра на чистом JavaScript

## Описание

В рамках курсовой работы по дисциплине «WEB-технологии» на чистом JavaScript разработана игра с 
видом  сверху, котором игроку нужно бродить в подземелье и сражаться с монстрами. Карта состоит из 
различных плиток на нескольких слоях, отображение происходит с помощью рисования картинок на 
хосте. Игра состоит из двух уровней с двумя типами врагов, которые отличаются атакой и моделью 
поведения. Для выбора ими направления движения применён модифицированный алгоритм A*. Также 
игра имеет звуковые эффекты, громкость которых зависит от расстояния источника до игрока, и анимации 
действий игрока и врагов.

## Стек технологий

Python, Django REST Framework, MongoDB, Nginx, Docker

## Запуск проекта
Находясь в директории репозитория выполнить:
-   ```
    docker build -t game .
    ```
-   ```
    docker run -p 3000:3000 game
    ```

Ссылка на сайт: http://127.0.0.1:3000
